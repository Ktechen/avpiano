# **PianoFlow**

## Beschreibung

"PianoFlow" ist eine Web-Anwendung zum Erstellen von eigenen Soundtracks. Mit Hilfe des Desings und den eingebundenen Features soll das kreieren von Musik leicht und spielerisch werden.
